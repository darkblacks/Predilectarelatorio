# Dashboard Predilecta

Apresentação web em React + Vite baseada na estrutura do projeto Transmassa.

## Como rodar

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Dados

O dashboard lê automaticamente o arquivo:

```txt
public/data/carregamento_predilecta.xlsx
```

Abas do XLSX:

- `Mensal`: lançamento mensal no formato Mês / Transportadora / Cliente / QTD de transportes.
- `Diario`: evolução diária de junho da unidade Predilecta.
- `Metas`: meta de terceiros usada nos gráficos.
- `Resumo`: indicadores auxiliares para conferência.
